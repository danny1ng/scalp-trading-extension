# Extension Icons

## Quick Setup

For now, the extension will work without icons (Chrome will use default).

To add proper icons, you can:

1. **Use icon.svg as reference** - Convert it to PNG at different sizes
2. **Use any online tool** like:
   - https://cloudconvert.com/svg-to-png
   - https://convertio.co/svg-png/
3. **Or use design tools** like Figma, Photoshop, etc.

## Required Files

- `icon16.png` - 16x16px (toolbar icon)
- `icon48.png` - 48x48px (extension management)
- `icon128.png` - 128x128px (Chrome Web Store)

## Icon Design

The provided `icon.svg` uses:
- Background: `#22c55e` (green, matching the app theme)
- Icon: TrendingUp from lucide (white stroke)
- Size: 128x128px with 24px rounded corners

Simply export this SVG at the 3 required sizes.

## Alternative: Skip Icons

The extension works perfectly without icons - Chrome will show a default puzzle piece icon.
