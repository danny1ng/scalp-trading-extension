# Contributing

## Documentation Update Rule
When developing this extension, keep user-facing documentation current.

Update `README.md` whenever you change:
- user-visible behavior
- supported exchanges/domains
- setup/install steps
- usage flow
- payload structure
- limitations or safety notes

## Scope of README
`README.md` is intended for end users first.  
Keep implementation details minimal there unless they directly affect user setup or usage.
